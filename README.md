# ISTPET: Modernización de Sistema de Gestión de Escuela de Conducción

Este proyecto representa la modernización integral del sistema ISTPET, transformando un esquema de base de datos relacional en una arquitectura de software desacoplada, escalable y automatizada.

## Ecosistema Tecnológico
- **Backend**: .NET 8 (Web API) con Entity Framework Core y AutoMapper.
- **Frontend**: React 18 + Vite + Tailwind CSS.
- **Base de Datos**: MySQL (11 Tablas, Vistas y Procedimientos Almacenados).
- **DevOps**: CI/CD Pipelines con GitHub Actions.
